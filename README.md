# TensorWire Homebrew Tap

```bash
brew tap tensorwire/tap
```

## Packages

### ai — GPU-Accelerated ML CLI

```bash
brew install tensorwire/tap/ai
```

Train, infer, quantize, and serve LLMs. Zero Python, one binary.

```bash
ai pull Qwen/Qwen2.5-0.5B
ai chat Qwen2.5-0.5B
ai serve Qwen2.5-0.5B
```
